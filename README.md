# CodeAlpha - Basic Network Sniffer

## 📌 Project Overview

This project is a simple network sniffer developed using Python and Scapy.
It captures network packets and displays useful information such as source IP, destination IP, protocol type, and payload data.

## 🚀 Features

* Capture live network packets
* Display Source IP and Destination IP
* Detect TCP and UDP protocols
* Analyze packet payloads
* Simple command-line interface

## 🛠️ Technologies Used

* Python
* Scapy Library

## ▶️ How to Run

### 1. Install Scapy

```bash
pip install scapy
```

### 2. Run the Program

```bash
python network_sniffer.py
```
